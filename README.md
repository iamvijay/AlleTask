# Screenshots Gallery App

An iOS app built using **Swift**, **UIKit**, and the **MVVM architecture** for viewing and managing screenshots from the photo library.

## Features
- **Thumbnail and Enlarged Views**: Displays screenshots in a carousel format.
- **Optimized Caching**: Uses `PHCachingImageManager` for efficient memory usage.
- **Dynamic Layout**: Includes a custom flow layout with scaling for center cells.
- **Smooth Navigation**: Supports seamless scrolling and synchronization between thumbnail and enlarged views.
- **Efficient Memory Management**: Dynamically caches and releases images to ensure optimal performance.
