# Screenshots Gallery App

An iOS app for viewing and managing screenshots from the photo library with optimized caching, smooth scrolling, and dynamic asset management.

## Features
- **Thumbnail and Enlarged Views**: Displays screenshots in a carousel format.
- **Optimized Caching**: Uses `PHCachingImageManager` for efficient memory usage.
- **Prefetching**: Implements `UICollectionViewDataSourcePrefetching` for smooth scrolling.
- **Dynamic Layout**: Includes a custom flow layout with scaling for center cells.
